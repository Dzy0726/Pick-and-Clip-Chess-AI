# 挑夹棋

## 1. 项目背景

该项目是同济大学软件学院2018级C语言的课程设计项目，要求编写一个能给出挑夹棋下一步走法的程序。对战平台、信息交互和基本的程序框架是助教完成。





## 2. 项目文件

|   文件名    |        备注        |
| :---------: | :----------------: |
|   main.c    | 初始示例代码及注释 |
|   final.c   |    最终提交代码    |
| 挑夹棋final |  Google Camp讲解   |
|  Pre.pptx   |      答辩PPT       |
